PRIMARY GOAL:
Users can upload a tattoo design (PNG/SVG), capture or import a short body-only video (5–8s), place/scale/rotate the design, and export a preview video where the tattoo looks like ink on skin.    - Validate file types: PNG/JPG/WEBP for photos, MP4/MOV for videos, PNG/SVG for designs.
   - Enforce max sizes: design <= 10MB, video <= 50MB, image <= 15MB.

4) Determinism:

- Feature flag: ENABLE_SORA=true
- Endpoint: POST /v1/ai/video-generate

- Must be resilient: if OpenAI returns 401/403/404 or model unavailable -> fallback.
